# עולם המדע האינטראקטיבי — עמוד לומדות

עמוד שמרכז 12 לומדות מדע אינטראקטיביות, בעברית (RTL), בנוי להטמעה כ-iframe באתר Wix.

---

## 📁 מה יש בתיקייה

```
index.html              ← העמוד הראשי (גרסת ריבוי-קבצים)
embed-single-file.html  ← קובץ יחיד עצמאי (הכול מוטמע בפנים, כולל התמונות)
assets/
  ├── styles.css        ← כל העיצוב
  ├── app.js            ← רקע החלקיקים והאנימציות
  └── tweaks.js         ← פאנל הכוונון (מופיע רק בעריכה, מוסתר בהטמעה)
images/
  └── 01.webp … 12.webp ← 12 תמונות הלומדות (יחס 4:3)
```

יש שתי דרכים להשתמש — בחרי אחת:

---

## דרך א׳ — העלאת כל התיקייה (מומלץ)

מעלים את **כל** הקבצים תוך שמירה על מבנה התיקיות (`index.html` בשורש, ולצידו `assets/` ו-`images/`).

### העלאה ל-GitHub והפעלת GitHub Pages
1. צרי ריפו חדש ב-GitHub (לדוגמה `science-hub`).
2. גררי לתוכו את כל תוכן התיקייה הזו (כולל `assets/` ו-`images/`).
3. ב-GitHub: **Settings → Pages**.
4. תחת *Build and deployment → Source* בחרי **Deploy from a branch**, ענף `main`, תיקייה `/ (root)`, ושמרי.
5. אחרי דקה-שתיים תתקבל כתובת כמו:
   `https://<שם-המשתמש>.github.io/science-hub/`

> חשוב: קישור רגיל לקובץ ב-GitHub (`github.com/...`) **לא** יוצג כדף. חייבים את כתובת ה-Pages (`github.io`).

---

## דרך ב׳ — קובץ יחיד

`embed-single-file.html` עצמאי לחלוטין — התמונות מוטמעות בתוכו, אין תלות ב-`assets/` או `images/`.
אפשר להעלות **רק אותו** (גם דרך GitHub Pages) ולקבל כתובת ישירה אליו.

---

## 🔗 הטמעה ב-Wix

1. ערכי את הדף ב-Wix → **Add (+) → Embed → Embed a Site / iFrame** (רכיב "HTML iframe").
2. בחרי **Website address**.
3. הדביקי את כתובת ה-GitHub Pages שקיבלת.
4. מתחי את הרכיב לרוחב המלא, וקבעי גובה נדיב (הדף מסתגל אוטומטית לרוחב; מומלץ להתחיל מ-1600px גובה ולכוונן).
5. שמרי ופרסמי.

> העמוד נבנה בטוח להטמעה: ללא `position:fixed`, ללא `position:sticky`, וללא גלילה לעוגן — כך שהוא מתנהג נקי בתוך iframe של Wix.

## 🖼️ החלפת תמונה בעתיד

כל לומדה משתמשת בתמונה מתיקיית `images/` ביחס **4:3**.
כדי להחליף: שמרי תמונה חדשה באותו שם (למשל `images/03.webp`) או עדכני את ה-`src` של אותו `<img class="shot">` ב-`index.html`.
לשמירה על מראה אחיד — איור line-art זוהר על רקע כהה, נושא יחיד במרכז, בצבעי המותג.
